\# Aegis-AI: Thermal \& Fan Controller



\[English](#english) | \[Türkçe](#türkçe)



---



\## English



\### 🚀 Overview

Aegis-AI is a high-performance Windows utility designed for real-time thermal monitoring and intelligent hardware control using AI. It monitors CPU and GPU metrics to optimize fan speeds and prevent thermal throttling.



\### ✨ Key Features

\- \*\*Multi-Source Monitoring:\*\* Real-time data for CPU (WMI) and NVIDIA GPU (NV-SMI).

\- \*\*AI Anomaly Detection:\*\* Uses \*\*LSTM\*\* models to predict unusual thermal patterns.

\- \*\*Dynamic Fan Control:\*\* Custom RPM adjustments based on workload.

\- \*\*Hardware Protection:\*\* Automated safety protocols for high-end components like the RTX 5060.



\### 🛠 Tech Stack

\- Python, C#, PyTorch, ONNX, WMI, NVIDIA-SMI.

\## Türkçe



\### 🚀 Genel Bakış

Aegis-AI, yapay zeka kullanarak gerçek zamanlı sıcaklık takibi ve akıllı donanım kontrolü sağlayan, yüksek performanslı bir Windows sistem aracıdır. CPU ve GPU verilerini izleyerek fan hızlarını optimize eder ve performans düşüşlerini (throttling) engeller.



\### ✨ Temel Özellikler

\- \*\*Çok Kaynaklı Takip:\*\* CPU (WMI) ve NVIDIA GPU (NV-SMI) için gerçek zamanlı veri toplama.

\- \*\*Yapay Zeka ile Anomali Tespiti:\*\* Beklenmedik sıcaklık artışlarını tahmin etmek için \*\*LSTM\*\* modelleri kullanır.

\- \*\*Dinamik Fan Kontrolü:\*\* İş yüküne göre özelleştirilebilir RPM ayarları.

\- \*\*Donanım Koruması:\*\* RTX 5060 gibi üst düzey bileşenlerin ömrünü korumak için otomatik güvenlik protokolleri.



\### 🛠 Teknoloji Yığını

\- Python, C#, PyTorch, ONNX, WMI, NVIDIA-SMI.
